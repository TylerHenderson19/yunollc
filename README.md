## SMS Notification Opt-In

### Emergency Alert System

Our SMS notification system provides real-time emergency alerts to property managers for urgent building maintenance issues.

### How Property Managers Opt In

Property managers opt in during service setup and configuration:

1. During onboarding, property managers provide their contact phone number
2. Property managers explicitly consent to receive emergency SMS alerts for their managed properties
3. We configure the notification preferences based on their requirements
4. Property managers confirm their enrollment and test the system

By opting in, you agree to receive text messages regarding:
- Building emergencies (flooding, gas leaks, electrical hazards, etc.)
- Urgent maintenance requests requiring immediate attention
- Critical property issues

### Message Frequency

Messages are sent only when legitimate building emergencies occur. Typical frequency: 0-5 messages per week depending on property conditions.
